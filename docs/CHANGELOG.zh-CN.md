# 变更日志

本文件记录项目的重要变化。

当前项目仍处于 1.0 之前的开发阶段。正式版本发布前，主要变化会先记录在 `Unreleased` 中。

## Unreleased

### 新增

- 受 lazygit 启发的 TUI 布局，支持数字快捷区域。
- 类目和子类目导航。
- 支持搜索类目、子类目和具体工具。
- JSON 工作台，支持美化、压缩、校验、排序、转义、反转义、路径查询和输出链式应用。
- 输入和输出工作区增加明显边框。
- 用于常用命令的 Actions 区域。
- 分组命令帮助弹窗，支持过滤。
- 输入和输出半页滚动快捷键。
- 输入编辑器撤销和重做。
- 核心内置工具 registry 级测试。
- 开源项目基础文档：README、贡献指南、行为规范、安全策略和变更日志。

### 变更

- README 改为更符合开源项目规范的文档结构。
- 通用工具 UI 可以根据工具是否需要输入自适应。
- JSON 类目使用单一工作台页面，不再拆成多个原子 JSON tab。

### 修复

- 复制、保存和 pipe 操作使用 trim 后的原始输出，不再使用 viewport 渲染文本。
- 长输入和长输出按面板宽度自动换行。
- JSON 校验快捷键不再与粘贴快捷键冲突。
